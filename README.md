# INHA_Algorithm_003

## Build and Run Instructions

1. Create a build directory:
   ```bash
   mkdir build && cd build
   ```

2. Compile your source code:
   ```bash
   cmake ..
   ```

3. Run make:
   ```bash
   make
   ```

4. Execute the program:
    ```bash
    ./runner
    ```

5. Etc./ shortcut command:
   ```
   cmake .. && make && ./runner
   ```
